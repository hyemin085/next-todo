# 🎀 TodoList(오늘할일)

 단조롭게 등록만 하는게 아닌 자기가 원하는 5가지 색에서 고른 메모에</br>
 간단하고 짧게 할일을 적어 놓고 </br>
 체크표시 통해 내가 해낸 일, 앞으로 해야 할 일의 구별이 가능하고</br>
 또한 자신이 어떤 일을 기록해 놨었는지에 대한 검색기능 또한 제공됩니다.</br>
 짧게 짧게 할일을 적어보고 작심삼일에서 벗어나 오늘할일을 시작해봐요!!</br>
